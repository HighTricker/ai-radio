# 我的歌单 / 偏好

> 复制本文件为 `taste.md` 后填入你自己最爱的歌（每行一首）。
> 格式：`歌名 - 歌手`
> 后端会把这些歌作为「最低保底候选」喂给 LLM 推荐器，配合年度歌单一起构成你的听歌画像。
> 找不到的歌会自动跳过并提示，不会阻塞播放。
>
> 建议放 9-30 首你「百听不腻」的歌；想做精细分组（心情 / 时段 / 主题）见 V2.x 扩展。
